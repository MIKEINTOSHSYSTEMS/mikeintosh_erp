Overview

First you need to go to employee settings to select source,destination and operation type after that go to products screen from custody app to choose which products to be available in custody by activate is property choice then go to user groups to select 'custody transfer' to specify the stock users that can manage internal transfer

Features

Configure some settings by route to employee
Create custody request for Employees
Access rights for Manager & User
Make some products available in custody by check property in product view
New request make transfer sent to Inventory Manager to complete delivery
Can Return Custody to the company inventory
